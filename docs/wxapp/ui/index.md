# 小程序端 / UI组件库

## NutUI Vue Taro 🔥

NutUI-Vue 组件库，基于 Taro，使用 Vue 技术栈开发小程序应用，开箱即用，帮助研发快速开发用户界面，提升开发效率，改善开发体验。

- 默认语言：简体中文
- 官方文档：https://nutui.jd.com/taro/vue/4x/#/zh-CN/guide/intro
- 仓库资源：https://github.com/jdf2e/nutui
